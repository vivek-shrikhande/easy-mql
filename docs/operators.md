# Operators

Following table lists all the operators supported by EasyMQL along with their arity and associativity.
The rows are ordered in the decreasing order of their precedence.

| Operator(s)      | Arity | Associativity |
| :--------------: | :---: | :-----------: |
| `NOT`            | `1`   | &xlarr;       |
| `* / %`          | `2`   | &xrarr;       |
| `+ -`            | `2`   | &xrarr;       |
| `< <= > >= = !=` | `2`   | &xrarr;       |
| `AND OR`         | `2`   | &xrarr;       |
